#### 隐式转换

无符号不能装有符号的，有符号的如果范围涵盖就可以装无符号的

浮点数可以装载任何类型的整数的，反之不可

### 异常捕获

```c#
try{
可能失败的内容

}
catch(Exception e){
如果出错，则执行
    Exception e 存储错误信息
}
//可选部分
finally{
    最后执行的代码，无论对错
}
```
